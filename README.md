💖 Valentine's Yes/No Game 💖
A fun and interactive Valentine's Day web game where you ask the big question: "Will you be my Valentine?"

🎮 How It Works
If the user clicks "Yes", they are taken to a special love-themed page.

If the user clicks "No", the button starts moving around the screen, making it harder to say no! 😆

Each time "No" is clicked, the game displays a sad message and changes the button image.

🛠 Tech Stack
HTML, CSS, JavaScript

Google Fonts & GIF animations

Interactive button movements & dynamic UI changes

Perfect for a fun Valentine's surprise.

Simple yet engaging interaction.
